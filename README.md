# AzerothLabWorks Addons

WotLK 3.3.5a addons used by AzerothLabWorks projects.

## Addons

- `GMCommandCenter`: AzerothCore GM workflow helper.
- `HybridTalentUI`: Addon-first Hybrid spell/talent interface for `AzerothLabWorks/azerothcore-hybrid-lab` dev branch `codex/hybrid-talents-ui`.

## Install

Copy the addon folder you want into your WoW client:

```text
World of Warcraft 3.3.5a\Interface\AddOns\<AddonFolder>
```

Example:

```text
C:\Games\WoW-3.3.5a-HD-Dev\Interface\AddOns\HybridTalentUI
```

Restart WoW or reload the UI, then enable the addon from the AddOns button on the character select screen.

## Matching Server Requirements

`GMCommandCenter` requires an AzerothCore account with the GM permissions needed for the commands you run.

`HybridTalentUI` requires the matching Hybrid Talent System server module from:

```text
https://github.com/AzerothLabWorks/azerothcore-hybrid-lab/tree/codex/hybrid-talents-ui
```
